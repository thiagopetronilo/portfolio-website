GLOBAL-NETWORK-3D

// OVERVIEW

Global-Network 3D is an immersive WebGL visualization environment that renders real-time telemetry traffic across a digital holographic globe. Designed to eliminate cognitive fatigue when monitoring complex decentralized systems, it maps data packet trajectories directly onto spatial spherical coordinates.

// CORE FEATURES

  • Spatial Wireframe Globe: Built using matrix physics computations to project a lightweight vector topology of Earth natively in the browser.
  • Reactive Points System: Employs 1,200 mathematical point coordinates memory-optimized via React Hooks to simulate live server nodes.
  • Orbit Telemetry Controls: Integrated 360-degree interaction mechanics allowing seamless mouse drag and pan perspective changes.

// ENGINEERING STACK

React, Three.js, React Three Fiber, React Three Drei, Vite, and Lucide Icons.

CLOUD-COST-MATRIX

// OVERVIEW

Cloud-Cost Matrix is a high-performance FinOps (Cloud Financial Operations) SaaS dashboard designed to bridge the gap between heavy infrastructure metrics and intuitive financial decisions. The main purpose is to eliminate dashboard fatigue for DevOps engineers and financial analysts by condensing multi-cloud expenses into an actionable, real-time interactive terminal.

// CORE FEATURES

  • Dynamic Simulation Engine: Built a reactive infrastructure forecasting tool using state-driven computation. Moving the slider recalculates projected costs across AWS, GCP, and Azure natively.

  • Interactive Telemetry: Integrated analytical stream tracking with custom visual depth, allowing users to cross-examine historical multi-cloud spend trends.
